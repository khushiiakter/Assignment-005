Important FAQ

What is the Document Object Model (DOM)?

DOM means Document Object Model. It has three things- html, css and javascript.All this things create DOM. DOM makes the html like an object in javascript, so that we can make interactions with javascript. This object is a model of the whole html.By using DOM, we can get all the elements of html, makes so many changes, add many actions.DOM makes the experience better between users and the website.

How do you select an element from the DOM?

We can select an element from the DOM in many ways in javascript. There are many methods like - getElementById(), getElementsByClassName(),getElementsByTagName(),querySelector(),querySelectorAll() etc. First we write document then apply use a method to call the element. Example - document.getElementById('Element name');

What is event delegation in the context of the DOM, and why is it useful?

HTML events are "things" that happen to HTML elements. When JavaScript is used in HTML pages, JavaScript can "react" on these events. For example:
1.The user selects, clicks, or hovers the cursor over a certain element.
2.The user chooses a key on the keyboard.
3.The user resizes or closes the browser window.
4.A web page finishes loading.
5.A form is submitted.
6.A video is played, paused, or finishes.
7.An error occurs.
